Documentation Page for myFlix Movie API

In this project, I will be builing the server-side component of a movie database web application. This web application will provide users with access to information about different movies, directors, and genres. Users can sign up, update their personal info, and create lists of all their favorite movies.

We will be creating a REST API using the MERN stack. (MongoDB, Express, React, and Node.JS)

Request URL HTTP Method Request Body Response Body
Return a list of ALL movies to the user /movies GET None JSON object holding data about all movies
Return data about a single movie by title to the user /movies/[ID] GET None JSON object holding data about selected movie
Return data about a genre by name/title /genres GET None Returns information about genres selected
Return data about a director by name /directors/[ID] GET None Returns JSON object about selected director
Allow new users to register /users POST JSON object holding data about new user:
Username: req.body.Username,
Password: req.body.Password,
Email: req.body.Email,
Birthday: req.body.Birthday
 		Returns JSON in this format:
ID: Integer
Username: String
Password: String
Email: String
Birthday: Date
Allow users to update their user info /users/:Username PUT JSON object holding data to be updated:
Username: req.body.Username,
Password: req.body.Password,
Email: req.body.Email,
Birthday: req.body.Birthday
 		Returns JSON in this format:
Username: String (required)
Password: String (required)
Email: String (required)
Birthday: Date
