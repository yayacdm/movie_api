Documentation Page for myFlix Movie API

In this project, I will be builing the server-side component of a movie database web application. This web application will provide users with access to information about different movies, directors, and genres. Users can sign up, update their personal info, and create lists of all their favorite movies.

We will be creating a REST API using the MERN stack. (MongoDB, Express, React, and Node.JS)

Request URL HTTP Method
Return a list of ALL movies to the user /movies GET
Return data about a single movie by title to the user /movies/[ID] GET
Return data about a genre by name/title /genres GET
Return data about a director by name /directors GET
Allow new users to register /users POST
Allow users to update their user info /users/[ID] PUT
Allow users to add a movie to their list of favorites /users/favs POST
Allow users to remove a movie from their list of favorites /users/favs DELETE
Allow existing users to deregister /users DELETE
